# OWCS - Claro

Este projeto usa:

* HTML e JavaScript ( jQuery ) tradicional
* SASS
* Bootstrap ( Grid )
* BrowserSync ( Gulp - NPM Module )

## Pré-requisitos

* [Git](https://git-scm.com/)
* [Node](http://nodejs.org/)
* [Gulp](http://gulpjs.com/)
* [Bower](http://bower.io/)

## Executando Local

```sh
$ gulp dev
```

## Deploy para produção

```sh
$ gulp build
```
